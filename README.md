# Composer
An android project for learning. Using a combination of Dagger Hilt, Android Compose, Room, Retrofit, and Robolectric unit tests.

This app demonstrates chatting between Android devices using socket connection. First, it uses Android Network Service Discovery (NSD) to find host and client IP address and port. Then, based on the address information both devices will connect with socket. Finally user can send chats by sending data to input/output stream.
